## Installation

Clone the project

```
git clone git@github.com:devsargam/Instagram-Clone-Backend.git
```

Setup the necessary environment variables

```
copy .env.example .env
```

Run the migrations with

```
npx prisma migrate dev
```

Run the server

```
pnpm start:dev
```
